### Альтернативный способ создания приложения React с использованием тулинга Vite

Приложение также можно создать с помощью инструмента Vite. Документация по созданию приложения [React](https://vitejs.dev/guide/)

1. Откройте терминал и введите следующую команду: yarn create vite my-app --template react, либо yarn create vite my-app --template react-ts, если вам нужен шаблон с TypeScript. Эта команда создаст настроенный шаблонный проект.
Откройте созданный проект в своей IDE.
Установите зависимости.
Готово. Чтобы запустить приложение, введите команду: yarn dev (либо npm run dev).
