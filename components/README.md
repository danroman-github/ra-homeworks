# Компоненты

## Задание

Научится создавать функциональный и class-based компоненты для реализации страницы интернет-магазина;

Компоненты — это фундаментальные строительные блоки любого React-приложения, похожие на детали конструктора. Научившись работать с компонентами, вы сможете создавать интерактивные и масштабируемые интерфейсы из небольших, независимых и переиспользуемых частей.

## Описание задания

1. [Страница интернет-магазина (функциональный компонент).](ra-store-func/my-vite-project)

2. [Страница интернет-магазина (class-based компонент).](ra-store-class/my-vite-project)
